if(조건문) 여러가지 조건식을 구성

다중 if문 필요 예: - 10>5 & 6>2 & 3>1

if(조건식) {
//실행문장1
}else if(조건식) {
\n//실행문장2
5 }else{
//실행문장3
}




switch(조건문)여러가지 연속된 값을 비교

case문: switch문에 있는 값과 비교 후 실행하는 구문

- 정수, 문자열, 논리타입의 값으로 구성

break문: 다른 case문으로 넘어가지 않도록 종료하는 구문

- 경우에 따라 생략 가능

default문: 모든 case문이 일치하지 않은 경우에 동작하는 구문

생략가능 ( = else문)
